# 🐱 Meow.js

**The cutest cat-themed animation & interaction library for JavaScript.**

Turn any website into a chaotic and adorable cat playground! 🐾

![Version](https://img.shields.io/badge/version-1.1.0-pink)
![License](https://img.shields.io/badge/license-MIT-blue)

## Features

- `chaseLaser()` – Follows your mouse like a laser pointer
- `zoomies()` – 3AM cat energy
- `catnip()` – Total chaos mode
- `pounce()`, `tailWag()`, `stretch()`, `loaf()`, `knockOver()`
- `boop()` – Click to show floating paw prints
- `carry()` – Drag and carry the element like a real cat
- `pawPrints()` & `fishRain()` – Fun particle effects
- Real meow sound effects

## Quick Start

```html
<script src="https://cdn.jsdelivr.net/gh/Uzairbaig123/meow.js/meow.js"></script>

<script>
  Meow.tailWag("#my-cat");
  Meow.boop("#my-cat");
  Meow.pawPrints();
</script>
